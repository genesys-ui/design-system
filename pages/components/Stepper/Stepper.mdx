# Stepper

The purpose of the Stepper component is to provide a visual representation of a multi-step process

## Overview

The component can display the status of each step, indicating whether it is completed, in progress, or disabled, and allows the user to navigate between steps using buttons.

## Accessibility

Recommended attributes to be included:

### aria-label

The [aria-label](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-label)
on the container with the value "progress" helps provide important context to screen readers.

### aria-current

The [aria-current](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-current)
indicates the current step on the list items.

### aria-hidden

The [aria-hidden](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-hidden)
excludes the separators elements from the screen readers.

### role="img" + aria-label="steps separator"

Using [ARIA: img role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/img_role#using_roleimg_to_confer_meaning_that_is_obscured_or_implied)
in combination with `aria-label` will confer meaning to the arrow icons between items.
