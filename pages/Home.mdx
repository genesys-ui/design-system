# Genesys Design System

Genesys is Devo's design system for products and digital ecosystems and home
to the foundations, human interface guidelines, design tools, UI components and
interaction patterns that we use to build great experiences.

Get started with a collection of tools and guides to assist in the development process for users.

Devo maintains the design kit in Figma, containing Genesys components. We support React framework implementation
through the component library called Genesys UI, and we have several visual libraries to support the usage of icons,
HTML + CSS styles and design tokens
